# Методы строк в Python

| Метод | Описание |
|-------|----------|
| `capitalize()` | Преобразует первый символ строки в заглавную букву, остальные делает строчными. |
| `casefold()` | Преобразует строку в нижний регистр (агрессивное преобразование, подходит для безрегистрового сравнения). |
| `center()` | Возвращает строку, выровненную по центру, дополненную указанным символом до заданной ширины. |
| `count()` | Возвращает количество непересекающихся вхождений указанного значения в строке. |
| `encode()` | Возвращает строку, закодированную в байты с использованием указанной кодировки. |
| `endswith()` | Возвращает `True`, если строка заканчивается указанным значением (суффиксом). |
| `expandtabs()` | Заменяет символы табуляции (`\t`) пробелами с заданным размером табуляции. |
| `find()` | Ищет в строке указанное значение и возвращает индекс первого вхождения. Возвращает -1, если не найдено. |
| `format()` | Форматирует строку, подставляя переданные значения в поля замены `{}`. |
| `format_map()` | Форматирует строку, используя словарь (mapping) для подстановки значений в поля замены. |
| `index()` | Ищет в строке указанное значение и возвращает индекс первого вхождения. Вызывает `ValueError`, если не найдено. |
| `isalnum()` | Возвращает `True`, если строка не пуста и все символы в ней — буквы или цифры (без пробелов и знаков). |
| `isalpha()` | Возвращает `True`, если строка не пуста и все символы в ней — буквы. |
| `isascii()` | Возвращает `True`, если строка не пуста и все символы принадлежат набору ASCII. |
| `isdecimal()` | Возвращает `True`, если строка не пуста и все символы — десятичные цифры (0–9). |
| `isdigit()` | Возвращает `True`, если строка не пуста и все символы — цифры (включая надстрочные и другие варианты). |
| `isidentifier()` | Возвращает `True`, если строка является допустимым идентификатором Python (имя переменной, функции и т.п.). |
| `islower()` | Возвращает `True`, если в строке есть хотя бы одна буква и все буквы — в нижнем регистре. |
| `isnumeric()` | Возвращает `True`, если строка не пуста и все символы — числовые (цифры, дробные символы, римские и т.п.). |
| `isprintable()` | Возвращает `True`, если строка пуста или все символы — печатные (не являются escape-последовательностями). |
| `isspace()` | Возвращает `True`, если строка не пуста и состоит только из пробельных символов. |
| `istitle()` | Возвращает `True`, если строка оформлена как заголовок (каждое слово с заглавной, остальные строчные). |
| `isupper()` | Возвращает `True`, если в строке есть хотя бы одна буква и все буквы — в верхнем регистре. |
| `join()` | Объединяет элементы итерируемого объекта в одну строку, вставляя между ними исходную строку-разделитель. |
| `ljust()` | Возвращает строку, выровненную по левому краю, дополненную символами справа до заданной ширины. |
| `lower()` | Преобразует строку к нижнему регистру. |
| `lstrip()` | Возвращает копию строки с удалёнными ведущими (слева) пробельными символами (или указанными символами). |
| `maketrans()` | Возвращает таблицу перевода, используемую в методе `translate()`. |
| `partition()` | Разделяет строку по первому вхождению разделителя и возвращает кортеж `(до, разделитель, после)`. |
| `replace()` | Возвращает строку, в которой все вхождения одного значения заменены на другое (можно ограничить количество замен). |
| `rfind()` | Ищет значение с конца строки и возвращает индекс последнего вхождения. Возвращает -1, если не найдено. |
| `rindex()` | Ищет значение с конца строки и возвращает индекс последнего вхождения. Вызывает `ValueError`, если не найдено. |
| `rjust()` | Возвращает строку, выровненную по правому краю, дополненную символами слева до заданной ширины. |
| `rpartition()` | Разделяет строку по последнему вхождению разделителя и возвращает кортеж `(до, разделитель, после)`. |
| `rsplit()` | Разбивает строку по разделителю, начиная справа, и возвращает список (можно ограничить количество разбиений). |
| `rstrip()` | Возвращает копию строки с удалёнными хвостовыми (справа) пробельными символами (или указанными символами). |
| `split()` | Разбивает строку по разделителю и возвращает список (можно задать максимальное количество разбиений). |
| `splitlines()` | Разбивает строку по символам перевода строки и возвращает список строк без самих символов разрыва. |
| `startswith()` | Возвращает `True`, если строка начинается с указанного значения (префикса). |
| `strip()` | Возвращает копию строки с удалёнными начальными и конечными пробельными символами (или указанными). |
| `swapcase()` | Меняет регистр каждого символа: строчные становятся заглавными, заглавные — строчными. |
| `title()` | Преобразует первый символ каждого слова в заглавный, остальные буквы слова — в строчные. |
| `translate()` | Возвращает строку с заменами символов согласно таблице перевода (полученной из `maketrans()`). |
| `upper()` | Преобразует строку к верхнему регистру. |
| `zfill()` | Дополняет строку нулями слева до указанной длины (учитывает знак числа, если он есть). |

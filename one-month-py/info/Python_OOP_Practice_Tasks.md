## 13. Конвертер валют (Метод класса)
Уровень: Средний
Создайте класс Currency. Добавьте атрибут класса:

usd_rate

change_rate()

который изменяет текущий курс валюты.

## 14. Университет
Уровень: Средний
Создайте класс University.
Атрибут класса:

* country (страна)

Атрибуты экземпляра:

* name (название)
* students_count (количество студентов)

Методы:

* show_info() — выводит информацию об университете.
* метод класса change_country() — изменяет страну для всех университетов.

## 20. Транспортная система
Уровень: Сложный
Создайте базовый класс Transport.
Поля:

* brand (марка)
* speed (скорость)

Метод:

* move() — движение.

Создайте три дочерних класса:

* Car (Автомобиль)
* Bus (Автобус)
* Bicycle (Велосипед)

Каждый дочерний класс должен иметь один уникальный атрибут и один уникальный метод.
Пример:
## Car

* fuel (топливо)
* refuel() (заправить)

## Bus

* passengers (пассажиры)
* pick_up() (подобрать пассажиров)

## Bicycle

* gears (передачи)
* ring_bell() (позвонить в звонок)

Создайте по два объекта каждого класса и продемонстрируйте работу всех методов.

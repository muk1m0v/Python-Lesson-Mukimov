# Easy Level

## List Tasks:
1. **Find the sum of all elements in a list.**  
   Write a function that takes a list of numbers and returns their sum.

2. **Check if an element is in the list.**  
   Write a function that checks if a given number is in the list.

3. **Create a list of squares of numbers.**  
   Write a program that takes a list of numbers and returns a new list containing the squares of those numbers.

## Dictionary Tasks:
1. **Count occurrences of elements in a dictionary.**  
   Write a program that counts how many times each number appears in a list and stores them in a dictionary.

2. **Check if a key exists in a dictionary.**  
   Write a function that checks if a key exists in a dictionary.

3. **Invert keys and values in a dictionary.**  
   Write a program that takes a dictionary and returns a new one with keys and values swapped.

# Medium Level

## List Tasks:
1. **Remove all even numbers from a list.**  
   Write a function that removes all even numbers from the list.

2. **Merge two lists without duplicates.**  
   Write a program that merges two lists and removes any duplicates from the result.

3. **Find the largest and smallest element in a list.**  
   Write a program that finds the largest and smallest element in a list.

## Dictionary Tasks:
1. **Get keys with values greater than a given number.**  
   Write a program that takes a dictionary and a number and returns all keys with values greater than the given number.

2. **Merge two dictionaries.**  
   Write a program that merges two dictionaries. If keys overlap, add their values together.

3. **Create a dictionary with the count of vowels in each string.**  
   Write a program that takes a list of strings and creates a dictionary where the keys are the strings and the values are the count of vowels in each string.

---

# Hard Level

## List Tasks:
1. **Find all sublists of a specific length in a list.**  
   Write a program that finds all possible sublists of a specified length in a given list.

2. **Find all numbers that appear more than once.**  
   Write a program that finds all numbers that appear more than once in the list and returns them in a new list.

3. **Sort the list based on the sum of digits of the elements.**  
   Write a program that sorts a list of numbers based on the sum of the digits of each number.

## Dictionary Tasks:
1. **Group elements of a list by their key in a dictionary.**  
   Write a program that takes a list of tuples, where the first element is the key and the second element is the value, and groups the values by their keys in a dictionary.

2. **Count the most frequent word in a text.**  
   Write a program that takes a string of text and counts the most frequent word in the text.

3. **Find all keys whose values are lists in a dictionary.**  
   Write a program that finds all keys in a dictionary whose values are lists.

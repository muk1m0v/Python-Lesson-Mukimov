
# 🧠 Практика по Python: Замыкания (Closures) — Набор №2

> ## 📚 Что нужно сделать
> Для каждого задания создайте **функцию**, которая **возвращает другую функцию**.
> Используйте **замыкания**, а не классы.

---

# 🟢 Начальный уровень

## 1. 🎫 Генератор билетов
### Создайте функцию:
```python
create_ticket_generator()
```
**Требования:**
- возвращает функцию;
- нумерация начинается с **1001**;
- каждый вызов увеличивает номер на **1**.

**Пример**
```text
1001 → 1002 → 1003
```

---

## 2. 💰 Калькулятор налога
Создайте `create_tax_calculator(tax_percent)`.

**Внутри замыкания хранится:**
- процент налога.

**Возвращаемая функция:**
- принимает цену;
- возвращает цену с налогом.

---

## 3. 📝 Форматирование имени
Создайте `create_name_formatter(prefix)`.

Возвращаемая функция:
- принимает имя;
- возвращает строку:

```text
Mr. + John → Mr. John
```

---

## 4. ➕ Сложение чисел
Создайте `create_adder(number)`.

Возвращаемая функция:
- принимает число;
- прибавляет сохранённое значение.

---

## 5. ❤️ Счётчик лайков
Создайте `create_like_counter()`.

Каждый вызов:
- увеличивает количество лайков на 1;
- возвращает текущее значение.

---

# 🟡 Средний уровень

## 6. 🏦 Банковский счёт
Создайте `create_account(initial_balance)`.

Возвращаемая функция:
- принимает сумму пополнения;
- обновляет баланс;
- возвращает новый баланс.

---

## 7. 📧 Имитация отправки Email
Создайте `create_email_sender(sender_email)`.

Возвращаемая функция:
- принимает email получателя;
- принимает сообщение;
- выводит информацию об отправке.

---

## 8. 🎮 Счёт в игре
Создайте `create_score_tracker()`.

Возвращаемая функция:
- принимает заработанные очки;
- суммирует их;
- возвращает общий счёт.

---

## 9. 🚀 Ограничитель запросов API
Создайте `create_rate_limiter(max_requests)`.

Функция должна:
- считать количество запросов;
- разрешать запросы до достижения лимита;
- после превышения возвращать сообщение об отказе.

---

## 10. 🛒 Учёт товаров
Создайте `create_inventory(initial_stock)`.

Возвращаемая функция:
- принимает количество проданных товаров;
- уменьшает остаток;
- возвращает текущий остаток.

---

### 16. Intersection with Count / Пересечение с количеством вхождений

#### **Problem / Задача**
Given multiple lists, find elements that appear in all lists, considering the number of occurrences.

Дано несколько списков. Найдите элементы, которые встречаются во всех списках, с учетом количества вхождений.

**Example / Пример**  
**Input:**  
```python
list1 = [1, 2, 2, 3, 4]
list2 = [2, 2, 3, 5]
list3 = [2, 3, 3, 6]
```
**Output:**  
```python
[2, 3]
```
(Только `2` и `3` встречаются во всех списках хотя бы один раз.)

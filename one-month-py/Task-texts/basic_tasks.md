# Python Tasks (English & Russian)

## Easy Level / Лёгкий уровень

### 1. Even or Odd? / Чётное или нечётное?
**Task:** Write a program that asks the user for a number and prints whether it is even or odd.

**Задача:** Напишите программу, которая запрашивает у пользователя число и определяет, является ли оно чётным или нечётным.

**Example:**
```
Input: 7
Output: Odd

Input: 4
Output: Even
```

---

### 2. Simple Password Checker / Простой проверяющий пароль
**Task:** Ask the user for a password. If the password is "python123", print "Access granted"; otherwise, print "Access denied".

**Задача:** Запросите у пользователя пароль. Если пароль — "python123", вывести "Access granted", иначе вывести "Access denied".

**Example:**
```
Input: python123
Output: Access granted

Input: secret
Output: Access denied
```

---

### 3. Count Vowels in a Word / Подсчёт гласных в слове
**Task:** Ask the user for a word and count how many vowels (a, e, i, o, u) it contains.

**Задача:** Запросите у пользователя слово и посчитайте, сколько в нём гласных букв (a, e, i, o, u).

**Example:**
```
Input: hello
Output: 2

Input: programming
Output: 3
```

### 4. Multiples of Three / Числа, кратные трём
**Task:** Print all numbers from 1 to 30 that are divisible by 3.

**Задача:** Выведите все числа от 1 до 30, которые делятся на 3.

**Example:**
```
Output: 3, 6, 9, 12, 15, 18, 21, 24, 27, 30
```

---

### 5. List Reverser / Переворот списка
**Task:** Given a list of numbers, print the reversed version of the list.

**Задача:** Данный список чисел, выведите его в обратном порядке.

**Example:**
```
Input: [1, 2, 3, 4, 5]
Output: [5, 4, 3, 2, 1]
```

## Medium Level / Средний уровень

### 6. FizzBuzz
**Task:** Print numbers from 1 to 50, but:
- If a number is divisible by 3, print "Fizz".
- If a number is divisible by 5, print "Buzz".
- If a number is divisible by both 3 and 5, print "FizzBuzz".

**Задача:** Выведите числа от 1 до 50, но:
- Если число делится на 3, вывести "Fizz".
- Если число делится на 5, вывести "Buzz".
- Если число делится и на 3, и на 5, вывести "FizzBuzz".

**Example:**
```
Output: 1, 2, Fizz, 4, Buzz, Fizz, 7, 8, Fizz, Buzz, 11, Fizz, 13, 14, FizzBuzz, ...
```

---

### 7. Palindrome Checker / Проверка на палиндром
**Task:** Ask the user for a word and check if it is a palindrome (the same forward and backward).

**Задача:** Запросите у пользователя слово и проверьте, является ли оно палиндромом (читается одинаково в обоих направлениях).

**Example:**
```
Input: madam
Output: Yes, it is a palindrome

Input: hello
Output: No, it is not a palindrome
```

---

### 8. Find the Maximum / Найдите максимум
**Task:** Given a list of numbers, find and print the maximum number **without using** `max()`.

**Задача:** Дан список чисел, найдите и выведите максимальное число **без использования** `max()`.

**Example:**
```
Input: [10, 4, 7, 15, 3]
Output: 15
```

---

### 9. Remove Duplicates / Удаление дубликатов
**Task:** Given a list of numbers, remove all duplicates and print the result.

**Задача:** Данный список чисел, удалите все дубликаты и выведите результат.

**Example:**
```
Input: [1, 2, 2, 3, 4, 4, 5]
Output: [1, 2, 3, 4, 5]
```

### 10. Letter Frequency Counter / Подсчёт частоты букв
**Task:** Ask the user for a sentence and count how many times each letter appears.

**Задача:** Запросите у пользователя предложение и посчитайте, сколько раз встречается каждая буква.

**Example:**
```

Output: h:1, e:1, l:3, o:2, w:1, r:1, d:1
```

**Задача:** Реализуйте шифр Цезаря, который сдвигает каждую букву строки на 3 позиции вперёд.


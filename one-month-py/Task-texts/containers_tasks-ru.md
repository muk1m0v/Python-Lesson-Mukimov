# Легкий уровень

## Задачи со списками:
1. **Найти сумму всех элементов списка.**  
   Напишите функцию, которая принимает список чисел и возвращает их сумму.

2. **Проверить, есть ли элемент в списке.**  
   Напишите функцию, которая проверяет, содержится ли заданное число в списке.

3. **Создать список квадратов чисел.**  
   Напишите программу, которая принимает список чисел и возвращает новый список, содержащий квадраты этих чисел.

## Задачи со словарями:
1. **Подсчитать количество вхождений элементов в словаре.**  
   Напишите программу, которая подсчитывает, сколько раз каждое число встречается в списке, и сохраняет результат в словаре.

2. **Проверить существование ключа в словаре.**  
   Напишите функцию, которая проверяет, существует ли ключ в словаре.

3. **Поменять местами ключи и значения в словаре.**  
   Напишите программу, которая принимает словарь и возвращает новый словарь с поменянными местами ключами и значениями.

# Средний уровень

## Задачи со списками:
1. **Удалить все чётные числа из списка.**  
   Напишите функцию, которая удаляет из списка все чётные числа.

2. **Объединить два списка без дубликатов.**  
   Напишите программу, которая объединяет два списка и удаляет все дубликаты из результата.

3. **Найти наибольший и наименьший элемент в списке.**  
   Напишите программу, которая находит наибольший и наименьший элемент в списке.

## Задачи со словарями:
1. **Получить ключи со значениями больше заданного числа.**  
   Напишите программу, которая принимает словарь и число и возвращает все ключи, значения которых больше заданного числа.

2. **Объединить два словаря.**  
   Напишите программу, которая объединяет два словаря. Если ключи совпадают, сложите их значения.

3. **Создать словарь с количеством гласных в каждой строке.**  
   Напишите программу, которая принимает список строк и создаёт словарь, где ключами являются строки, а значениями — количество гласных в каждой строке.

# Сложный уровень

## Задачи со списками:
1. **Найти все подсписки заданной длины в списке.**  
   Напишите программу, которая находит все возможные подсписки указанной длины в данном списке.

2. **Найти все числа, которые встречаются более одного раза.**  
   Напишите программу, которая находит все числа, встречающиеся в списке более одного раза, и возвращает их в новом списке.

3. **Отсортировать список по сумме цифр элементов.**  
   Напишите программу, которая сортирует список чисел по сумме цифр каждого числа.

## Задачи со словарями:
1. **Сгруппировать элементы списка по ключу в словаре.**  
   Напишите программу, которая принимает список кортежей, где первый элемент — ключ, а второй — значение, и группирует значения по их ключам в словаре.

2. **Подсчитать самое частое слово в тексте.**  
   Напишите программу, которая принимает строку текста и подсчитывает самое часто встречающееся слово.

3. **Найти все ключи, значениями которых являются списки.**  
   Напишите программу, которая находит все ключи в словаре, значения которых являются списками.

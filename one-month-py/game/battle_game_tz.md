# Техническое задание: Игра «Бой с боссом» 🐉

## Что мы делаем

Консольная пошаговая игра. Игрок создаёт своего персонажа, выбирает ему оружие, а потом сражается с боссом. Бой идёт **по раундам**: каждый проход цикла — это один раунд, в котором сначала бьёт игрок, потом босс. Цикл длится, пока у кого-то не закончится здоровье (HP). После каждого удара в консоль выводится, сколько урона нанесено и сколько HP осталось.

Все данные о персонаже, боссе и оружии хранятся в **словарях (dict)**.

---

## Этап 1. Словарь персонажа

Создай словарь, который описывает героя. В нём должны быть такие ключи:

- `name` — имя персонажа (строка)
- `type` — класс/тип (например `"Warrior"`, `"Mage"`)
- `hp` — здоровье (число)
- `attack` — сила атаки (число)
- `defense` — защита (число)
- `weapon` — название оружия (строка)

Пример того, как должен выглядеть готовый словарь:

```python
hero = {
    "name": "Aragorn",
    "type": "Warrior",
    "hp": 100,
    "attack": 20,
    "defense": 5,
    "weapon": "sword"
}
```

**Задача этапа:** сделать так, чтобы значения можно было либо задать вручную, либо запросить у игрока через `input()` (имя, тип, оружие — по желанию).

---

## Этап 2. Словарь оружия с бафами

Создай отдельный словарь, в котором описано **всё доступное оружие**. У каждого оружия свои бонусы (бафы): прибавка к атаке и/или к защите.

```python
weapons = {
    "sword":  {"attack_bonus": 10, "defense_bonus": 2},
    "axe":    {"attack_bonus": 15, "defense_bonus": 0},
    "staff":  {"attack_bonus": 5,  "defense_bonus": 8},
    "dagger": {"attack_bonus": 8,  "defense_bonus": 4},
}
```

Это **словарь словарей**: по ключу `"sword"` лежит ещё один словарь с его бонусами.

**Задача этапа:** взять оружие героя (`hero["weapon"]`), найти его в `weapons` и **прибавить бонусы** к его атаке и защите.

```python
chosen = weapons[hero["weapon"]]
hero["attack"] += chosen["attack_bonus"]
hero["defense"] += chosen["defense_bonus"]
```

boss = {

**Задача этапа:** так же, как у героя, прибавить боссу бонусы от его снаряжения перед началом боя.

## Этап 4. Формула урона

Нужно решить, сколько урона проходит за один удар. Простая и понятная формула:

```
урон = атака_бьющего - защита_того_кого_бьют
```

Важное правило: урон не может быть отрицательным. Если защита больше атаки, урон должен быть минимум `1` (или `0` — реши сам), иначе HP будет случайно **расти**.

```python
damage = attacker_attack - defender_defense
if damage < 1:
    damage = 1
```

**Задача этапа:** написать этот расчёт. Можешь оформить его как функцию `calculate_damage(attacker, defender)`, которая принимает два словаря и возвращает число урона.

## Этап 5. Главный цикл боя (раунды)

Это сердце игры. Используй цикл `while`, который крутится, **пока оба живы** (HP больше 0):

```python
while hero["hp"] > 0 and boss["hp"] > 0:
    # 1. Герой бьёт босса
    # 2. Выводим: сколько урона нанёс герой, сколько HP у босса осталось
    # 3. Проверяем — если босс умер (hp <= 0), выходим из цикла (break)
    # 4. Босс бьёт героя
    # 5. Выводим: сколько урона нанёс босс, сколько HP у героя осталось
```

Каждый проход этого `while` — это **один раунд**. Порядок внутри раунда:

1. Герой атакует → у босса уменьшается HP.
2. Печать результата удара героя.
3. Проверка: если босс мёртв — `break` (босс не должен бить после смерти).
4. Босс атакует → у героя уменьшается HP.
5. Печать результата удара босса.

**Задача этапа:** реализовать этот цикл с понятным выводом каждого удара.

## Этап 6. Подсчёт раундов и итог боя

Заведи счётчик раундов **до** цикла и увеличивай его в начале каждого раунда:

```python
round_number = 0
while hero["hp"] > 0 and boss["hp"] > 0:
    round_number += 1
    print(f"--- Раунд {round_number} ---")
    # ... удары ...
```

После завершения цикла определи победителя и выведи итог: кто победил и за сколько раундов.

```python
if hero["hp"] > 0:
    print(f"{hero['name']} победил за {round_number} раундов!")
else:
    print(f"{boss['name']} оказался сильнее...")
```

**Задача этапа:** добавить счётчик и финальное сообщение о победе/поражении.

- **Шанс крита.** С вероятностью, скажем, 20% удар наносит двойной урон. Выводи в консоль «КРИТ!».
- **Несколько боссов.** Сделай список боссов и проходи их по очереди — победил одного, выходит следующий.
- **Зелья лечения.** Дай герою пару зелий, которые он может использовать, чтобы восстановить HP в раунде.

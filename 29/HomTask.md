## ҲАЛЛИ МАСЪАЛАҲО:
### 1) Ҷамъбасти элементҳо
TJ: Рӯйхати ададҳоро гиред ва суммаи ҳамаи элементҳоро ҳисоб кунед (бо while).
RU: Дан список чисел. Найдите сумму всех элементов (через while).

Input: 
```
1 2 3 4
```
Output: 
```
10
```
### 2) Шумораи элементҳо
TJ: Дар рӯйхат чанд элемент ҳаст?
RU: Сколько элементов в списке?

Input: 
```
5 6 7
```
Output: 
```
3
```

## 3) Калонтарин элемент
TJ: Калонтарин ададро ёбед (бо while).
RU: Найдите максимальный элемент (while).

Input: 
```
3 9 2 5
```
Output: 
```
9
```

## 4) Хурдтарин элемент
TJ: Хурдтарин ададро ёбед.
RU: Найдите минимальный элемент.

